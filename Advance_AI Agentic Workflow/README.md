# 🤖 Tata Group — Agentic AI Platform

A production-grade Streamlit application demonstrating **Multi-Agent AI** workflows
across 5 enterprise business use cases.

---

## 🚀 Quick Start

```bash
pip install -r requirements.txt
streamlit run agentic_ai_app.py
```

---

## 📋 5 Business Use Cases

| # | Use Case | Agents | Key Features |
|---|----------|--------|--------------|
| 1 | 🎓 Learning & Development | 9 | Skill gap, learning path, certification tracker |
| 2 | 🧑‍💼 Recruitment | 8 | Resume screening, funnel analytics, candidate ranking |
| 3 | 🖥️ IT Service Desk | 7 | Auto-resolution, diagnostics, SLA monitoring |
| 4 | 🛒 Procurement | 9 | Vendor discovery, RFQ, cost savings calculator |
| 5 | 🌴 Leave Management | 9 | Policy validation, team calendar, auto-approval |

---

## ✨ Bonus Features
- **Agent Execution Timeline** — Gantt-style parallel agent visualisation
- **AI Reasoning Log** — Decision engine transparency
- **Leaderboard** (L&D) — Company-wide learning rankings
- **ROI Analytics** — Cost savings and impact metrics
- **Auto-Approval Logic** — Policy-driven autonomous decisions
- **Downloadable Reports** — Text-based agent execution reports
- **Dark Theme UI** — Tata brand-aligned enterprise aesthetic

---

## 🏗️ Architecture

```
User Input → Orchestrator
                ├── Parallel Agents (Phase 1)
                ├── Sequential Agents (Phase 2)
                └── Notification Broadcast
```

Built with: Streamlit · Plotly · Pandas
